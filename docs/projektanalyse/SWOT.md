SWOT Bereich	Beispiel im Projekt
Stärken	• Hotel besitzt bestehende IT-Infrastruktur
• Klare Vision zur Digitalisierung
• Motiviertes internes IT-Team
Schwächen	• Alte Server (4 Jahre+), evtl. Leistungslimits
• Personal braucht evtl. Schulung fürs neue System
• Prozesse noch nicht vollständig digital dokumentiert
Chancen	• Personalisierte Gästeinfos steigern Zufriedenheit
• Automatisierung spart Zeit an Rezeption
• Image-Modernisierung des Hotels
Risiken	• Gäste akzeptieren System evtl. langsam
• IT-Ausfälle wirken direkt auf Betrieb
• Lieferanten-Kommunikation muss angepasst werden
